# Challenge 08: GitHub Collab Quest (Hard)

## Objective
Practice GitHub collaboration workflow with pull requests.

## Files
- `team-log.md`

## Task
1. Create branch `docs/team-log-update`.
2. Add your name and one workshop learning in `team-log.md`.
3. Commit:
   - `challenge-08: update team log`
4. Push the branch to GitHub.
5. Open a Pull Request into `main`.
6. Ask a classmate for review.
7. Merge PR using Squash Merge.

## Success Criteria
- Branch is pushed.
- PR exists with review comment.
- Final merge commit on `main` is clean.
